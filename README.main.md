# frontforge

前端熔炉，bigfact 的前端实验室

## 目录

* src: 所有实验小项目
  * public: 公用文件

## 约定

* 每一次 git 提交，只涉及一个小项目内容
* 提交 git 备注格式，小项目名 + 版本 + 更新信息
* 类名大写
* sass 文件和 css 文件放在一个文件夹下

## 环境配置

* 安装

```
$ npm install
```

## 开发

```
$ npm run dev
```

## 发布

```
$ npm run build
```

## 代码相关

* 移动端调试引入 [vconsole](https://github.com/WechatFE/vConsole) (前端调试面板)
