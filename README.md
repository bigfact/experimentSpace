# frontforge

前端的实验环境

## 目录
* things: 所有实验小项目
* css: 公用的 css 文件
* js: 公用的 js 文件
* imgs: 公用的图片文件

## 环境配置
* 本地环境

> npm install

> bower install

* 全局环境

> npm install gulp (版本参照 package.json)

> gem install compass

## 开发

> gulp
