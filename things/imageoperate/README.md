# TinyThings - imageoperate
图片操作的组件

## 功能
    * 鼠标滚轮放大、缩小
    * 鼠标点击拖动图片
    * 图片翻转
    
## 参数
* config                // 默认配置
    * zoomTimes: 1,     // 缩放倍数
    * maxWidth: 1980,   // 最大宽度
    * minWidth: 320,    // 最小宽度
    * degree: 90,       // 每次旋转的度数
    * direction: 0      // 旋转方向及次数，正数为顺时针，负数为逆时针
